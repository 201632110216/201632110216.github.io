Introduction





What is javaScript?

JavaScript is the most popular programming language in the world. This language can be used for HTML and web, and can be widely used in servers, PC, laptops, tablet PCs and smart phones and other devices.


JavaScript is scripting language

JavaScript is a lightweight programming language.
JavaScript is a programming code that can insert HTML pages.
When JavaScript is inserted into the HTML page, it can be executed by all modern browsers.
JavaScript is easy to learn.


JavaScript: writes to the HTML output

For an example:

document.write("<'h1>This is a heading<'/h1>");
document.write("<'p>This is a paragraph<'/p>");

Hint: you can only use document.write in the HTML output. If you use this method after the document is loaded, it will overwrite the entire document.


JavaScript: react to events

For an example:

<'button type="button" onclick="alert('Welcome!')">Click here.

The alert () function is not always used in JavaScript, but it is very convenient for code testing.
The onclick event is just one of the many events you're going to learn in this tutorial.


JavaScript: changing HTML content

Using JavaScript to handle HTML content is a powerful feature.
For an example:

x=document.getElementById("demo") //Search element
x.innerHTML="Hello JavaScript"; //Change content

You'll often see document.getElementByID ("some ID"). This method is defined in HTML DOM.
DOM (document object model) is a formal W3C standard for accessing HTML elements.
You'll learn more about HTML DOM in more than one chapter of this tutorial.


JavaScript: change the HTML image

This meeting dynamically changes the origin of HTML <'image> (SRC):
The Light bulb
Click on the light bulb to turn on or off the light
JavaScript can change most of the attributes of any HTML element, not just pictures.


JavaScript: change the HTML style

Changing the style of HTML elements is a variant that changes the HTML attribute.
For an example:

x=document.getElementById("demo") //Search element
x.style.color="#ff0000"; //Change content


JavaScript is often used to validate user input.

For an example:

if isNaN(x) {alert("Not Numeric")};


Do you know?

JavaScript and Java are two completely different languages, both in concept and design.
Java (invented by Sun) is a more complex programming language.
ECMA-262 is the official name of the JavaScript standard.
JavaScript was invented by Brendan Eich. It appeared in Netscape in 1995 (the browser has stopped updating), and it was adopted by ECMA (a standard association) in 1997.
