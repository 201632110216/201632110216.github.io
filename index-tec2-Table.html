HTML Table





You can use HTML to create tables.

For an example:

<'html>
<'body>
<'p>Each table starts with the table tag.<'/p>
<'p>Each table row starts with the tr tag.<'/p>
<'p>Each table data starts with the td tag.<'/p>
<'h4>A row<'/h4>
<'table border="1">
<'tr>
<'td>100<'/td>
<'/tr>
<'/table>
<'h4>A row and three column :<'/h4>
<'table border="1">
<'tr>
<'td>100<'/td>
<'td>200<'/td>
<'td>300<'/td>
<'/tr>
<'/table>
<'h4>Two rows and three columns :
<'table border="1">
<'tr>
<'td>100<'/td>
<'td>200<'/td>
<'td>300<'/td>
<'/tr>
<'tr>
<'td>400<'/td>
<'td>500<'/td>
<'td>600<'/td>
<'/tr>
<'/table>
<'/body>
<'/html>


Table

The table is defined by the <'table> tag. Each table has a number of rows (defined by the <'tr> tag), and each row is divided into several cells (defined by the <'td> tag). The letter TD refers to the table data (table data), that is, the content of the data cell. Data cells can contain text, pictures, lists, paragraphs, forms, horizontal lines, tables, and so on.

<'table border="1">
<'tr>
<'td>row 1, cell 1<'/td>
<'td>row 1, cell 2<'/td>
<'/tr>
<'tr>
<'td>row 2, cell 1<'/td>
<'td>row 2, cell 2<'/td>
<'/tr>
<'/table>

The browser displays the following:
row 1, cell 1 row 1, cell 2
row 2, cell 1 row 2, cell 2


Table and border attributes

If the border property is not defined, the table does not display borders.
Sometimes this is useful, but most of the time we want to display borders.
Use border properties to display a table with a border:
<'table border="1"> <'tr> <'td>Row 1, cell 1<'/td> <'td>Row 1, cell 2<'/td> <'/tr> <'/table>


Table Header

The table header is defined with the <'th> tag.
Most browsers display headers in bold, centered text:
<'table border="1">
<'tr>
<'th>Heading<'/th>
<'th>Another Heading<'/th>
<'/tr>
<'tr>
<'td>row 1, cell 1<'/td>
<'td>row 1, cell 2<'/td>
<'/tr>
<'tr>
<'td>row 2, cell 1<'/td>
<'td>row 2, cell 2<'/td>
<'/tr>
<'/table>

Use border properties to display a table with a border:
Heading Another Heading
row 1, cell 1 row 1, cell 2
row 2, cell 1 row 2, cell 2
