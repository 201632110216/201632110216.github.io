HTML Image





You can display images in a document by using HTML.

For an example:

<'!DOCTYPE HTML>
<'html>
<'body>
<'p>
A picture:
<'img src="insect yellow.jpg" width="128" height="128" />
<'/p>
<'p>Notice that the syntax of inserting animated images is not different
from the syntax of inserting ordinary images.<'/p>
<'/body>
<'/html>


Image tags (<'img>) and source attributes (Src)

In HTML, the image is defined by the <'img> tag.
<'img> is an empty tag, which means that it contains only attributes, and there is no closing tag.
To display images on a page, you need to use source attributes (SRC). SRC means "source"". The value of the source property is the URL address of the image.
Defining the syntax of an image is:

<'img src="url" />


Replace text attributes (Alt)

The ALT property is used to define a bunch of ready, replaceable text for the image. The value of the replacement text attribute is user defined.
<'img src="boat.gif" alt="Big Boat">

When the browser cannot load images, replace text attributes to tell readers what information they have lost. At this point, the browser will display the alternative text instead of the image. Replacing text attributes on a page is a good habit, which helps to better display information and is very useful for people who use plain text browsers.


Basic considerations - useful tips:

If a HTML file contains ten images, then 11 files need to be loaded to show the page correctly. Loading pictures takes time, so My advice is: take care of pictures.
