CSS overview

1.CSS refers to cascading style sheets (Cascading Style Sheets)
2.The style defines how to display HTML elements
3.The style is usually stored in the stylesheet
4.Adding styles to HTML 4 is to solve the problem of separating content from performance
5. external style sheets can greatly improve work efficiency
6. external style sheets are usually stored in CSS files
7. more than one style definition can be cascaded into one



Style solves a common problem

The HTML tag was originally designed to define document content. By using <'h1'>, <'p'>, <'table'> such a label, HTML's intention is to express "is the title of this", "this is the end", "this is a table of such information. At the same time, the document layout is done by the browser without any formatting tags.
Because of the major browsers (Netscape and Internet Explorer) continue to be the new HTML tags and attributes (such as font tags and color attributes) added to the HTML specification, create the document content clearly is independent of the document presentation site becomes more and more difficult.
In order to solve this problem, the World Wide Web Consortium (W3C), a non-profit Standardization Alliance, undertake the HTML standardization mission, and create a style in HTML 4 (Style).
All mainstream browsers support cascading style sheets.


The stylesheet greatly improves work efficiency

The stylesheet defines how to display HTML elements, just as the font tags and color attributes of HTML 3.2 do. Styles are usually stored in external.Css files. By simply editing a simple CSS document, the external stylesheet enables you to change the layout and appearance of all the pages in the site at the same time. By allowing simultaneous control of the styles and layouts of multiple pages, CSS can be a breakthrough in the WEB design field. As a web developer, you can define styles for each HTML element and apply it to any number of pages you want. If you need a global update, simply change the style, and all elements in the site are automatically updated.


Multiple styles will be stacked into one

Style sheets allow you to specify style information in a variety of ways. The style can be specified in a single HTML element, in the head element of the HTML page, or in an external CSS file. You can even refer to multiple external style sheets within the same HTML document.


What style will be used when the same HTML element is defined by more than one style?

In general, all styles are stacked in a new virtual stylesheet based on the following rules, in which number 4 has the highest priority.
1.Browser default settings
2.External style sheet
3.Internal style sheet (located inside the <'head'> tag)
4.Inline style (within the HTML element)
Therefore, an inline style (in the inside of a HTML element) has the highest priority, which means that it will override a style statement below: style statement in <'head'> tags, a style statement in an external style sheet, or in a browser style statement (default).
